# 04. Nesting Components

**Date:** July 01, 2026

### What I Learned

Today I practiced **Nesting Components** — one of the most important concepts in Vue.js.

I created a small "Vue Ninjas" website where:
- `App.vue` is the parent component
- `Header.vue`, list of ninjas, and `Footer.vue` are child components

### Key Points:

- How to create child components
- How to import and register them in parent component
- How to use `components` option
- Component communication basics (for now just nesting)
- Scoped styles

### Main Conclusion:

Nesting components helps us break down big interfaces into small, reusable, and manageable pieces. This is the foundation of building real Vue applications.

This example includes:
- Header with gradient
- Clickable ninja cards (using `v-for` and `v-show`)
- Footer
